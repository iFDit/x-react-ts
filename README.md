# x-react-ts

### Simple React creater.
  - typescript
  - simple dependencies
  - out of the box

`````shell
  # install
  npm install --save-dev x-react-ts
`````

`````shell
  # initial a project.
  react-ts init [scope]
`````

`````shell
  # start webpack dev service.
  npm run start
`````

`````shell
  # build bundle
  npm run build:web
`````

`````shell
  # build lib(transform ts?x to js)
  npm run build:lib
`````
